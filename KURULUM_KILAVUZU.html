2
Firebase Kurulumu (Veritabanı)
1
Firebase Console'a git: console.firebase.google.com
2
Create a project → Proje adı: bina-yonetici → Devam et
3
Sol menüden Firestore Database → Create database → Standard edition → Next → Bölge: europe-west1 (Belgium) → Next
4
Start in test mode seç → Create butonuna bas
5
Veritabanı açılınca Rules sekmesine tıkla → Aşağıdaki kodu yapıştır → Publish bas:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }
⚠️ Bu kural herkese açık — güvenlik uygulama içindeki şifre sistemiyle sağlanıyor.
3

1
github.com → Giriş yap → Sağ üstte + → New repository

Add file → Upload files → ZIP içindeki 6 dosyanın tamamını sürükle bırak → Commit changes

Üstte Settings → Sol menüde Pages → Branch: main seç → Save → 2 dakika bekle → Sayfayı yenile
